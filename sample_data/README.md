# Sample Data

This directory contains publicly available demonstration images used for the VIGI Vision examples and CLI demonstrations.

These images are intended solely to demonstrate profile-based scene analysis and are **not** production surveillance images or real customer CCTV captures.

## Notes

- Public demonstration assets only.
- Real VIGI camera captures are stored locally under `private_data/` and are intentionally excluded from Git.
